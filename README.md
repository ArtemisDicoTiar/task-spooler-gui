# task-spooler GUI

A web-based GUI for task-spooler, for easy monitoring/access.

![Example of home page](./example.png)

# Setup

Install dependencies.

```bash
python3 -m venv .env && \
source activate.sh && \
pip install -r requirements.txt
```

# Run

Run these commands or use `tsp-gui`.

```bash
source activate.sh
flask run
```
